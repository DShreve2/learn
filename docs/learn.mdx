Netdata is a monitoring, visualization, and troubleshooting solution for systems, applications, and infrastructures of
all types. The free and open-source monitoring Agent works in collaboration with Netdata Cloud to offer both single-node
and infrastructure-wide monitoring in a single pane of glass. Collect everything, visualize metrics, troubleshoot
complex performance problems, and make Netdata's per-second metrics work with your existing monitoring stack, entirely
for free.
